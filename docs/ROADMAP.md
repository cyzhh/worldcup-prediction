## 如何拆 Issue

1. 在 GitHub 选择 **New Issue** → 使用模板 **Roadmap 任务**。
2. 标题格式：`[P1-8] 批量预测提速`。
3. 正文填写：背景、验收标准、关联文件、预估工时。
4. 里程碑：按 Sprint 分组（S2 / S3 / S4）。

模板文件：`.github/ISSUE_TEMPLATE/`
